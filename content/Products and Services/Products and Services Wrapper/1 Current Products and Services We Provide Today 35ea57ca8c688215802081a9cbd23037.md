# 1. Current Products and Services We Provide Today

## Managed IT Support

This is the core of what Stratum does today.

We support customers when they have technology issues. This includes remote support, onsite support, ticket work, troubleshooting, user support, vendor coordination, and communication around active issues.

### What this includes

- Remote troubleshooting
- Technical questions and user support
- Onsite visits
- Ticket-based support
- Escalations from junior techs to senior techs
- Client communication connected to active tickets
- Vendor coordination on behalf of customers
- Dispatch and scheduling support

### Brand lens connection

**Structure:** tickets, dispatch, priority, escalation, and accountability

**Stability:** keeping customers operational

**Simplicity:** one place to go when they need help

**Stewardship:** being a responsible technology partner today and into the future

---

## Server, Network, and Infrastructure Support

We also support the core systems that keep a customer’s business running.

This includes servers, networks, firewalls, Wi-Fi, switches, workstations, and other infrastructure that needs to be maintained, updated, documented, and replaced over time.

### What this includes

- Server maintenance
- Network maintenance
- Firewall and Wi-Fi support
- Patching and updates
- Health checks
- Infrastructure upgrades
- Network diagrams
- Client documentation
- Configuration notes and runbooks

### Brand lens connection

**Structure:** documented environments and organized systems

**Security:** properly configured networks and access

**Stability:** fewer outages and less reactive firefighting

---

## Backup and Recovery

We already do backup-related work, but it needs to become clearer as a product.

Customers need to know not only that they have backups, but that they can actually recover when something goes wrong.

### What this includes today

- Backup checks
- Backup verification
- Test restores
- Disaster recovery work
- Recovery after failure, ransomware, or data loss
- Backup troubleshooting

### Brand lens connection

**Security:** protecting data

**Stability:** keeping the business recoverable

**Stewardship:** making sure customers are protected today and recoverable into the future

---

## Security and Cybersecurity Support

We currently provide security and cybersecurity support, but it is not fully productized yet.

This needs to be clearer because security is not only about responding to cyber incidents. It also includes the basic security work that protects the customer every day: access control, passwords, MFA, devices, email, backups, patching, permissions, monitoring, and standards.

Some of this work happens through tickets, some through tools, some through monitoring, and some through contractor or specialist support.

### What this includes today

- Security incident response
- Suspicious activity investigation
- Malware or compromise response
- Account compromise support
- Security alert response
- MFA support
- Password and access control support
- Email security support
- Endpoint protection support
- Patch and update support
- Backup-related security protection
- User permission reviews
- Cybersecurity contractor support
- Monitoring and alert response

### Brand lens connection

**Security:** protecting the customer from risk

**Stability:** reducing disruption from cyber events

**Stewardship:** helping customers make better risk decisions today and building a safer path forward

---
