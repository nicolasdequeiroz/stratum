# 3. Stratum Secure

### Positioning

**Managed IT with stronger cybersecurity, backup, email/web protection, risk controls, and cyber-insurance support.**

### Target market

| Best Fit | Not Best Fit |
| --- | --- |
| Clients with cyber-insurance requirements | Client only needing basic support |
| Companies with sensitive data | Very small low-risk businesses |
| Clients with servers or business-critical Microsoft 365 use | Clients unwilling to invest in security controls |
| Businesses concerned about ransomware, downtime, or data loss | Clients only buying break-fix support |
| Growing clients needing security maturity | Clients needing full strategic partnership should move to Complete |
